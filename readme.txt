Пример запуска

python kvstorage.py storage.dat

--------------------------------------------------------
Взаимодействие:

- "key" = "value" Добавляет\обновляет пару в хранилище. Обязательно в кавычках
- ? key Достает значение по ключу
- ?? key Проверяет есть ли такой ключ
- x key Удаляет пару из хранилища
- close Закрытие хранилища

Для корректного завершения работы с хранилищем нужно закрыть его через
команду close без аргументов.
Есть возможность возобновления работы с хранилищем.
