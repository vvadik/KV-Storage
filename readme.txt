Пример запуска

python main.py my_account init

--------------------------------------------------------
Аргументы запуска 

python main.py [name] [command] [-k --key] [-v --value]

name - Имя хранилища
command - Действие с хранилищем. Поддерживаемые команды:
    1. init - создание хранилища
    2. list - вывод на консоль всего хранилища
    3. add [-k] [-v] - добавление или обновления пары ключ-значение
    4. delete [-k] - удаление пары ключ-значение
    5. load [-k] - вывод на консоль значения ключа
[-k] - Ключ
[-v] - значение
